### 技巧

#### **`toLocaleString()`** 函式用法

```js
var time = new Date();

//第一個參數：顯示語言＆格式
//第二個參數：option
console.log(time.toLocaleString("en-GB", {
  timeZone: "UTC",
  hour12: false,
}));
```

這次使用到的 Option 有兩個：

- timeZone
  - 可以接受像是 &#8220;America/New_York&#8221; 這樣的時區，參閱：[IANA time zone database][1]
- hour12
  - true //12小時制
  - false //24小時制
